# **App Name**: MonsterFinance

## Core Features:

- Secure Authentication: User authentication with Firebase for secure data access.
- Dashboard: Dashboard displaying key financial metrics and quick actions.
- Record Transactions: Record incomes and expenses.
- Reporting: Provide basic reports with income and expense tracking.
- Account Management: List accounts with balance to record transactions.
- Transacciones Management: Add Alivios or Transacciones from one account to another.
- AI-Powered Expense Suggestion: Recommend expense categories tool using generative AI to classify uncategorized expenses.

## Style Guidelines:

- Primary color: #A78BFA (a light purplish blue), for a modern tech-finance look
- Background color: #1E1B2A (very dark purple).
- Accent color: #E2E8F0 (light gray) for subtle highlights.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, neutral design.
- Use minimalist icons from Remix Icon or Feather icons.
- Use a simple grid layout with cards for displaying information.
- Subtle transitions and animations for a smooth user experience.
